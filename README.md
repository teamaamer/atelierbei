# Luxury Microblading Studio Website

A modern, elegant, and minimal website for a luxury microblading and permanent makeup studio.

## Design Features

- **Soft, feminine aesthetic** with warm ivory background and blush pink accents
- **Premium feel** with lots of white space and refined typography
- **Smooth animations** including fade-in effects and gentle floating elements
- **Fully responsive** design that works beautifully on all devices

## Color Palette

- Warm Ivory: `#F8F6F3`
- Soft Blush Pink: `#EAD6D9`
- Light Beige: `#EFEAE4`
- Muted Gold: `#C8A96A`

## Sections

1. **Hero Section** - Elegant introduction with image and call-to-action
2. **About Section** - Trust-building content about the artist
3. **Services Section** - Microblading, Powder Brows, Combination Brows, Touch-ups
4. **Before & After Gallery** - Showcase transformations
5. **Reviews Section** - Client testimonials with star ratings
6. **CTA Section** - Book appointment call-to-action
7. **Footer** - Contact information and social links

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm start
```

The website will open at [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm build
```

## Technologies Used

- React 18
- TailwindCSS 3
- Lucide React (icons)
- Custom animations and transitions
